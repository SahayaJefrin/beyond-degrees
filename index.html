Your toolkit

Tools & resources to start now

Prioritised — focus on MUST LEARN first. No need to learn everything at once.

Must Learn
🐍
Python
Core skill
The #1 language for data roles. Pandas, NumPy, Matplotlib, Seaborn — start here.
Where to start: Kaggle's free Python course → then practice on real datasets immediately. Don't just watch tutorials.
Must Learn
🗄️
SQL
Core skill
Every data job needs SQL. It's how you talk to databases — interviews almost always test this.
Where to start: SQLZoo → Mode Analytics SQL tutorial → HackerRank SQL challenges. Aim for 5-star rating like me!
Must Learn
📊
Data Visualisation
Core skill
Telling stories with data. Tableau, Power BI, Looker Studio, or just Matplotlib/Seaborn.
Where to start: Looker Studio is free + Google-backed. Build one dashboard from a public dataset and share it on LinkedIn.
Good to Have
🤖
Gen AI / LLMs
Emerging
The job market now expects you to know how to use AI tools — not just build models.
Where to start: Google's "Introduction to Generative AI" (free) → prompt engineering basics → try building a small LLM-powered tool.
Good to Have
🔁
Git & GitHub
Essential tool
Version control isn't optional anymore. Your GitHub is your live portfolio — every project goes here.
Where to start: GitHub's own beginner guide → commit your first Python script today. Recruiters check GitHub profiles.
Bonus Edge
🏆
Kaggle Competitions
Portfolio builder
Real datasets, real problems, real community. Even participating (not winning) signals drive to recruiters.
Where to start: Titanic survival prediction is the classic beginner comp. Join one this month — no pressure to win.
Free resources I actually used
Kaggle Learn
Free mini-courses on Python, SQL, ML, Data Viz. Bite-sized and practical. Best starting point.
Google Data Analytics Certificate
Coursera. Paid but often free with financial aid. Looks good on LinkedIn. Worth it in Year 2.
Towards Data Science (Medium)
Best articles written by practitioners. Read 2–3 articles a week to stay current with the industry.
LinkedIn + Naukri
Set up your profile now. Follow data leaders. Apply for internships even if you feel "not ready."
HackerRank SQL Track
Where I got my 5-star rating. Recruiters recognise it. Takes 2–3 weeks of consistent practice.
StatQuest (YouTube)
Josh Starmer explains ML and statistics visually. If you're confused by any concept — check here first.
Before you leave

Your 3-this-week action plan

Don't leave today motivated but undirected. Pick these three and do them this week.

01
Set up your GitHub today
Create an account, make a repo called "data-portfolio", push even one Python file. Your future self will thank you.
02
Start Kaggle's Python course
Free, takes 5 hours total. Do the first two modules this week. Consistency over intensity — 30 mins daily beats 5 hours on Sunday.
03
Update your LinkedIn profile
Add your education, add "Statistics" and "Python" as skills, connect with 10 people in data roles. Message one of them with a genuine question.
04
Apply for one internship
Even if you feel unready. Rejection teaches you what to improve. Naukri, LinkedIn, Internshala — search "data analyst intern" right now.
05
Join one data community
Kaggle forums, LinkedIn stats groups, DataTalks.Club Discord. Learning with others is 10x more effective than learning alone.
06
Find a project idea this week
Pick anything you're curious about — cricket stats, food delivery data, weather patterns. A project you care about is a project you'll finish.
